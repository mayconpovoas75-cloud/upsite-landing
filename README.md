# UP SITE

Landing page comercial da UP SITE, feita em React + Vite, com foco em gerar clientes pelo WhatsApp.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build de producao

```bash
npm run build
```

## Publicacao recomendada

Use Vercel para o deploy e a Hostinger apenas para o dominio.
